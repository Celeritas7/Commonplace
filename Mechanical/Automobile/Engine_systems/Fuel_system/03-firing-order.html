Overview

Firing order is the sequence in which the cylinders of a multi-cylinder engine fire one after another. It's chosen to balance crankshaft torque loading and minimise vibration. The notebook works through the crankshaft geometry, the stroke-state grid (Power / Exhaust / Suction / Compression abbreviated as P/E/S/C) per cylinder over the 720° four-stroke cycle, and gives standard firing orders for 4, 6, 8 and 12-cylinder engines.

Working Principle

How firing order is set by crankshaft geometry

A 4-stroke engine completes one full cycle every 720° of crank rotation (two revolutions per cycle, because the cam runs at half crank speed). With N cylinders, the firing events are spaced 720°/N apart along the crank rotation:

Cylinders Firing spacing Power overlap
4 720°/4 = 180° One power stroke per half-revolution; no overlap.
6 720°/6 = 120° Power overlap = 60°
8 720°/8 = 90° Power overlap = 90° (very smooth)
12 720°/12 = 60° Power overlap = 120° (extremely smooth)

More cylinders = more power events per crank revolution = smoother torque delivery.

The PESC grid

For any firing order, you can build a table: - Rows = crank-angle positions in 60° (or 90°) steps from 0° to 720°. - Columns = each cylinder (1 to N). - Cell = which of the four strokes that cylinder is in: Power, Exhaust, Suction (intake), Compression.

Walking the table cylinder by cylinder shows that at any moment, exactly one cylinder is doing a Power stroke in a 4-cyl engine, two cylinders in an 8-cyl, three in a 12-cyl — the smoothness gain.

Force Path / Process Flow

4-cylinder — inline-4 (firing order 1-3-4-2)

Notebook: "1-3-4-2" with sketch of crankshaft (180° crankpins between 1&4 / 2&3).

Crankshaft layout (180° between throws):
       1      4
       │      │
       ●──────●──────  crank axis
       │      │
       2      3

Firing order: 1 → 3 → 4 → 2 → (repeat)

Spacing: 180° between each fire event.

Why 1-3-4-2 and not 1-2-3-4? Adjacent cylinders firing back-to-back would put a heavy alternating bending load on the crankshaft. 1-3-4-2 spreads load symmetrically.

Notebook also lists 1-3-2-4 as an alternate spatial-distribution sketch — both 1-3-4-2 and 1-3-2-4 are valid inline-4 firing orders depending on crank design. Most production inline-4s use 1-3-4-2 (or its mirror 1-2-4-3).

6-cylinder — inline-6 (firing order 1-5-3-6-2-4)

Notebook: "1-5-3-4-2-6", "1-5-3-6-2-4", "1-3-2-4" (reads as a 4-cyl interpolation).

Crankshaft layout (120° between throws):
       1-6  ●  ◄── 0°
            │
       2-5  ●  ◄── 240°
            │
       3-4  ●  ◄── 120°

(Cylinders are paired on shared crankpins 1-6, 2-5, 3-4 in an inline-6.)

Firing order: 1 → 5 → 3 → 6 → 2 → 4 → (repeat)

Spacing: 120° between each fire event. 6 fire events per 720° cycle.

The classic BMW inline-6 firing order is 1-5-3-6-2-4 — it's why an I6 is famously smooth (perfect primary and secondary balance).

8-cylinder — V8 (firing order 1-8-4-3-6-5-7-2)

Notebook: "1-8-4-3-6-5-7-2" (most common), with alternates 1-5-3-7-8-2-6-4, 1-7-3-5-8-2-6-4, 1-7-3-4-8-2-6-5, 1-6-3-8-5-2-7-4.

Crankshaft layout (V8 cross-plane, 90° between throws):

    1    8

      ●  ◄── 0°
      │
    2    7

      ●  ◄── 90°  
      │
    3    6

      ●  ◄── 180°
      │
    4    5

      ●  ◄── 270°

Firing order: 1 → 8 → 4 → 3 → 6 → 5 → 7 → 2 → (repeat)

Spacing: 90° between each fire event. 8 fire events per 720° cycle.

1-8-4-3-6-5-7-2 is the most-used Detroit V8 firing order (also BMW V8 N62, BMW M62). Alternate orders shift which bank fires next.

12-cylinder — V12 (firing order 1-7-5-11-3-9-6-12-2-8-4-10)

Notebook IMG 112415 (Exhaust folder): "1-7-5-11-3-9-6-12-2-8-4-10" — full PESC table drawn.

Crankshaft: 6 crankpins at 60° intervals; two cylinders share each throw (one per bank).

Firing order: 1 → 7 → 5 → 11 → 3 → 9 → 6 → 12 → 2 → 8 → 4 → 10 → (repeat)

Spacing: 60° between each fire event. 12 fire events per 720° cycle.
Power overlap = 120° per cylinder → at any instant, 2 cylinders are mid-power.

This is the BMW M70/M73-era V12 firing order (S70 used a variant for the V12 LMR).

Spatial distribution tables (IMG 112117, top)

The notebook draws several 4×4 grids labelled with P/E/S/C — these are graphical PESC tables for 4-cyl engines. Reading down the table tells you what state each cylinder is in at the next firing event. The grids confirm that for a 1-3-4-2 firing order, at any 180° crank step the four cylinders are doing each of the four strokes — perfect overlap.

Components / Definitions

Term Meaning
Firing order Sequence in which cylinders fire in one full 720° cycle.
Firing spacing Crank-angle between consecutive fires = 720° / N (N = cyl count).
P / E / S / C Power / Exhaust / Suction (intake) / Compression — the four strokes.
Crank throw angle Angular offset between two crankpins on the crankshaft.
Power overlap How many degrees of crank rotation per cylinder where another cylinder is also mid-power.
Crossplane V8 V8 with 90° throws (1-8-4-3-6-5-7-2) — typical American/BMW V8.
Flatplane V8 V8 with 180° throws (Ferrari, Lotus).

Detailed Component Notes

Why firing order matters for the dealer technician

  • Wrong order = won't start or runs roughly with backfires.
  • After swapping spark plug leads / coils — confirm order matches the cylinder map in the service manual.
  • Misfire on cylinder N in a diagnostic trouble code → trace from the firing-order position, not just the cylinder number, when checking coil-pack / wasted-spark systems.

Power overlap and engine smoothness

Engine Power events per 720° Power overlap
1-cyl 1 none — vibration city
4-cyl 4 0° — every 180°
6-cyl I6 6 60° — already very smooth
8-cyl V8 8 90°
12-cyl V12 12 120° — silk

The notebook's stroke-grid tables are the trainer's way of teaching this visually: as you slide the firing order across more cylinders, the gaps between Power events shrink.

Why a cross-plane V8 isn't 1-2-3-4-5-6-7-8

The 90° crank throws don't put cylinders on adjacent crankpins next to each other in firing order — that would put two adjacent bores firing back-to-back and tear the crankshaft. 1-8-4-3-6-5-7-2 alternates between the two cylinder banks to balance load.

Key Facts

  • One 4-stroke cycle = 720° of crank → firing spacing = 720° / N.
  • Standard firing orders:
  • I4: 1-3-4-2 (alternate: 1-2-4-3 / 1-3-2-4)
  • I6: 1-5-3-6-2-4 (BMW inline-six)
  • V8: 1-8-4-3-6-5-7-2 (cross-plane V8, BMW M62 / N62)
  • V12: 1-7-5-11-3-9-6-12-2-8-4-10 (BMW M70 / M73)
  • More cylinders → smaller firing spacing → smoother engine (more power overlap).
  • The PESC grid lets you see at one glance how many cylinders are mid-power at any crank angle.
