Cricket player record

From my notes

For a player, the following data is maintained: name, batting average, bowling average, and number of catches taken. Write a program that inputs the data and prints it only if either the batting average is 40 or above, or the bowling average is 20 or above.

#include <iostream.h>
void main()
{
    char name[30];
    float batting, bowling;
    int catches;

    cout << "Enter the name";
    cin >> name;
    cout << "Enter the batting and bowling average";
    cin >> batting >> bowling;
    cout << "Enter no. of catches taken";
    cin >> catches;

    if (batting >= 40 || bowling >= 20)
        cout << name << batting << bowling << catches;

08

Largest element in a matrix

From my notes

Read an m × n matrix and find the largest element in it, using nested loops.

#include <iostream.h>
void main()
{
    int x[5][10], i, j, m, n, max;

    cout << "Enter rows and columns";
    cin >> m >> n;

    // input the matrix
    for (i = 0; i < m; i = i + 1)
        for (j = 0; j < n; j = j + 1)
            cin >> x[i][j];

    // find the largest
    max = x[0][0];
    for (i = 0; i < m; i = i + 1)
        for (j = 0; j < n; j = j + 1)
            if (x[i][j] > max)
                max = x[i][j];

    cout << max;
}
09

Multi-level inheritance — employee → teacher

From my notes

An educational institution maintains a database of its employees using classes. employee (name, id) is the base class; teacher (subject), typist (speed), and manager (title) are derived from it. Create classes with functions that let a user input data for different types of employees and display it. (Transcribed: the base employee and the derived teacher.)

#include <iostream.h>

class employee
{
protected:
    char name[30];
    int id;
public:
    void in()
    {
        cin >> name >> id;
    }
    void out()
    {
        cout << name << id;
    }
};

class teacher : public employee
{
    char subject[30];
public:
    void in()
    {
        employee::in();      // reuse the base class input
        cin >> subject;
    }
    void out()
    {
        employee::out();     // reuse the base class output
        cout << subject;
    }
};
