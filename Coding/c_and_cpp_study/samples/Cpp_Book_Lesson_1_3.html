Your First Program — hello.cpp

Five lines. We'll write them, click through every part, compile with g++, and run. By the end of this lesson you'll have your first running C++ program and you'll know what each piece does.

1 Why this matters

The rite of passage 🚀

A C++ program is a collection of commands or statements. Below is a simple code that has "Hello world!" as its output.

Your notes nailed it. Every C++ programmer's first program is "Hello, world!" — and going through it line by line teaches you almost the entire shape of how C++ source files are structured. After this lesson, you'll have read, written, compiled, and run real C++.

The pattern goes back to Brian Kernighan's 1972 tutorial (the same Kernighan who later co-authored the C book with Dennis Ritchie). Bjarne Stroustrup used it in his early C++ examples, and now every language does it first. You're joining a 50-year tradition.

2 The program

Five lines — meet your first C++ 💻

Below is the complete program. We'll click through every part next. For now, just read it once and notice the shape.
hello.cpp #include <iostream> int main() { std::cout << "Hello, world!\n"; return 0; }

Five lines (counting the blank one and the closing brace). That shape — #include at top, int main(), statements inside, return 0; at the end — is the bones of every C++ program. You'll write it thousands of times. Get the rhythm.

Notice something subtle that's different from a C program: we use std::cout << ... instead of printf(...). This is one of C++'s signature additions over C — the stream operator <<. We'll click into it in the next section.

4 Compile and run it

From source file to running program ⚡

Save the program above as hello.cpp (the extension matters — it must be .cpp, not .c). Open a terminal in that folder, then two commands: 
$ g++ -std=c++20 hello.cpp -o hello

That invokes the C++ compiler. The -std=c++20 flag tells it "use modern C++20" (per lesson 1.2). The -o hello says "name the output file hello." No output means no errors.

$ ./hello

That runs the executable. (On Windows cmd, you can type just hello without the ./.) Expected output:

✅ Expected output

Hello, world!

That's it — your first C++ program, running on real machine code. The C++ compiler turned your five lines into hundreds of bytes of optimized assembly, and your CPU just executed all of them in microseconds. Pretty good for "hello world."

6 Common mistakes

What trips beginners on their first C++ program 🚧

🙅
Forgetting the std:: in front of cout. cout lives inside the std namespace. Writing just cout << ... gives a "cout was not declared" error. Three fixes: write std::cout every time (what we do), add using namespace std; at the top (controversial — see common mistake #3 below), or add using std::cout; just for what you need (a clean middle ground).
🙅
Saving the file with the wrong extension. If you save your file as hello.c instead of hello.cpp, gcc will compile it as C, not C++ — and C doesn't have iostream or std::cout. You'll get bewildering errors about unknown identifiers. The fix: rename to .cpp and recompile with g++.
🙅
"using namespace std; — should I use it?" Your existing notes mention "the line using namespace std; tells the compiler to use the std namespace." True — but in modern C++, putting it at the top of files is considered bad practice for anything other than tiny snippets. It pulls thousands of names into your namespace, which causes name clashes in larger programs. We use the explicit std:: prefix throughout this book; that's the recommended modern style.

🎓 Quick recap

  • Every C++ program has the shape: #include at top, int main(), statements inside, return 0; at the end.
  • #include <iostream> gives you std::cout, std::cin, and friends — the standard input/output streams.
  • main() is the function the OS runs on launch. Only one of them per program.
  • std::cout << "text" prints. The << is the "insertion operator" — it streams data into cout. Chain as many as you like.
  • std:: is the namespace prefix. Modern C++ avoids using namespace std; at file scope.
  • Compile with g++ -std=c++20 hello.cpp -o hello, run with ./hello.
