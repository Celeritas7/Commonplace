📖 About these books

The books are designed for complete beginners — no prior programming experience assumed. Each lesson follows the same 7- or 8-section pattern: Why this matters → Mental model → Walk-through → Try it → Predict → Common mistakes → Exercises → Recap.

🟠 Have / Partial / New Each lesson marks which content comes from your existing notes (📖) and which is newly written (✨).
🎮 Playful but rigorous XP, streaks, confetti — but the content underneath is the same as any serious textbook.
📚 Independent books Read either book without the other. Some topics appear in both, written for that book's context.
Built one lesson at a time. To plan ahead, see TOPIC_LIST.md · For source materials, see C/ and C++/ folders
