An interactive book — every chapter is a live notebook you can edit and run in your browser.

How it works: Click any chapter to open it. Code runs entirely in your browser via Pyodide (no server, no install). Edit any cell and press Shift+Enter to re-run it and see the output change. Saved outputs are shown until you re-run. First load may take ~20-30s while the Python runtime downloads. An internet connection is required for the runtime and for any cells that fetch data from a URL.
